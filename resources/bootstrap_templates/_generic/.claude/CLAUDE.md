# CLAUDE.md — SCG Repo

This repository is part of the SupplyChainGuard (SCG) ecosystem.

## Non‑Negotiables

- SOLID / KISS / DRY. No speculative abstractions.
- No behavior changes without tests/docs.
- Prefer shared SCG libraries over random third‑party packages.
- Keep changes minimal, reviewable, and deterministic.

## Workflow

1. Read existing docs + conventions first.
2. Search locally (ripgrep) before guessing.
3. Provide diffs for edits and full content for new files.
